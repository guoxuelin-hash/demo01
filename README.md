# 网站收藏管理器

一个美观、功能完整的网站收藏管理器，支持添加、编辑、删除和搜索书签。

## 功能特性

- 📌 添加和管理网站收藏
- 🔍 搜索和筛选书签
- 📁 分类管理
- 💾 本地存储数据
- 🎨 现代化 UI 设计
- 📱 响应式布局

## GitHub Pages 部署

本项目已配置为可在 GitHub Pages 上运行。

### 启用 GitHub Pages

1. 访问仓库：https://github.com/guoxuelin-hash/demo01
2. 点击 **Settings（设置）**
3. 在左侧菜单中找到 **Pages（页面）**
4. 在 **Source（源）** 部分：
   - 选择 **Deploy from a branch（从分支部署）**
   - 选择分支：**master**
   - 选择文件夹：**/ (root)**
5. 点击 **Save（保存）**

### 访问地址

启用后，你的网站将在以下地址可访问：
- `https://guoxuelin-hash.github.io/demo01/`

通常需要几分钟时间部署，部署完成后你就可以通过上面的地址访问你的网站收藏管理器了！

## 本地运行

直接在浏览器中打开 `index.html` 文件即可运行。
